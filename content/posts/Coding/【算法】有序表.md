## 有序表的类型
- 红黑树 - Red Black Tree
- AVL - Adelson-Velsky and Landis 一种自平衡的搜索二叉树
- SB - Size Balanced Tree 节点大小平衡树

上述有序表的查找时间复杂度都为：$O\left(logn\right)$。

上述三种结构都属于BST - Binary Search Tree 系列。  
跳表 - Skiplist 和上述结构不属于一种类型。
